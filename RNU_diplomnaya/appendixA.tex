\chapter{Diagnostic Rule Specifications}
\label{app:rules}

This appendix provides the complete sensor threshold specifications for all five DTC rule groups implemented in \texttt{DiagnosticEngine}.
Each rule group is presented as a decision table mapping sensor conditions to cause hypotheses with assigned probability weights.
